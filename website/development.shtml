Development

SplitsBrowser is an open source Java Applet to view the results of orienteering events.

The code is freely available for non-commercial use under the GPL licence. This allows modification and redistribution of the source subject to the redistributed source also being made avaliable under a GPL licence agreement.

Source code is available as follows:

  • Current release (2.1.3)
  • 1.05 (Old style SplitsBrowser)

There is also an associated server side application written in php4 to allow organisers to submit results but no source has yet been released for this.

Possible extensions

  • Rewrite/refractor - The quality of the current code leaves a lot to be desired. The SplitsGraph and ControlCollection and SelectedResults classes are prime candidates to be refactored.
  • Map view to show courses and/or reconstructions.
  • Statistics view showing which controls people lost time etc.
  • Add import filter for IOF results format.
