Splitsbrowser input format help

Splitsbrowser supports the following formats:

SportIdent CSV format
Comma separated variable file output form SportIdent OE/MT program. This is the prefered format for SportIdent results. It includes start times which the are not generally present in the SportIdent HTML output files.

SportIdent HTML format
Use this format for SportIdent results when CSV format results are not avaliable. There are many variations to the layout of these results which means that it is not always read reliably. In particular if club names start with a number the file is not read correctly.

Splitsbrowser comma separated variable format
A simple comma separated variable format retained principally for compatibility with previous versions of Splitsbrowser.
